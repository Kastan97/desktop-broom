FAQ

Questions, answered

Is Desktop Broom really free?

Yes — Desktop Broom is 100% free forever. No ads, no sign-up, no premium tier. It's donation-supported freeware, so if you love it you can chip in, but you never have to.

How does it clean up a messy desktop?

Point Desktop Broom at your desktop or your Downloads folder and it automatically sorts every loose file into tidy folders by type — images, PDFs, documents and more. One click and your messy desktop is clean.

Is it safe? Will it delete my files?

Never. Desktop Broom does not delete anything — it only moves loose files into folders, and it never touches your existing folders or projects. It shows a full preview before doing anything, and there's one-click undo if you change your mind.

Do I need an AI key to use it?

No. The AI file organizer is completely optional and off by default. Basic desktop cleaning works with zero setup and zero AI. If you turn on Smart Mode to auto-rename files, you bring your own key (Claude, GPT or DeepSeek). Smart Mode sends only small text snippets to your chosen provider; your key stays local.

Can it organize my Downloads folder too?

Yes. Desktop Broom can declutter your desktop and organize your Downloads folder the same way, sweeping loose files into clean, labeled folders by type.

Which platforms does it support?

Desktop Broom runs on Windows on ARM (ARM64) today. An x64 build for regular Windows 10 and 11 PCs is coming soon, and an iOS version (sideload) is planned after that.
